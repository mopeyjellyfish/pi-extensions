# Debug loop evidence

Record a compact chain:

- exact reproduction command/input and observed failure;
- impact and affected boundary;
- hypotheses with predictions that could falsify each;
- isolating observation that proves root cause;
- regression test name and RED output reference;
- fix and GREEN output reference;
- focused/regression verification references;
- review result and residual risk.

Do not paste raw unbounded logs into the workflow ledger. Store safe paths or bounded command summaries. Redact credentials and personal data.

If the defect changes the pitch materially, rewind to Pitch rather than smuggling product scope into a bug fix. If it only reveals work inside the active slice, add that discovered work to Todo. If appetite expires, stop forward mutation and ask the user to resolve the circuit breaker.
